# Base Security Monitoring

A lightweight Python tool designed to capture essential system and network information for security auditing.

## Features
- System OS and Architecture detection.
- Network Hostname and IP retrieval.
- Error-resistant execution.

## Installation
1. Clone the repository:
bash
   git clone https://github.com/Younes-1919/base-security-monitoring.git

2.Navigate to the directory:
   cd base-security-monitoring
 
Usage

Run the monitor script using Python:
	
	python monitor.py
